How It Works

  • Paste a YouTube video, shorts, or playlist URL above
  • Video info is fetched using the noembed API (no key required)
  • Downloads are processed via cobalt.tools open-source API
  • If the API is unavailable, fallback links to download services are provided
  • Thumbnails can be downloaded directly in various resolutions
  • This tool respects YouTube's Terms of Service. Only download content you have rights to.
