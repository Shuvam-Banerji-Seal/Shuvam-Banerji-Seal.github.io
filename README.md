# Shuvam Banerji Seal — Portfolio

**Computational Chemist & AI Researcher**

Personal portfolio website showcasing research in computational chemistry, AI/ML, information retrieval, and full-stack engineering.

## 🔬 About

BS-MS student at IISER Kolkata (Chemistry Major, Computer Science Minor). Research spans DFT catalysis, RAG architectures, NLP/information retrieval, and molecular dynamics.

## 🚀 Features

- **Terminal-themed homepage** — Interactive shell with whoami, cat, ls commands
- **7 Dark Themes** — Random theme assignment on visit (Dark Coffee, Amber Night, Tokyo Night, Absolute Dark, Forest Night, Dracula, Molecular Terminal)
- **Music Player** — Spotify-style player with 450+ tracks
- **Tools Suite** — 13+ browser-based tools:
  - LLM Chat & Studio (OpenAI, Anthropic, Google Gemini, Groq)
  - PDF Studio (compress, merge, split, rotate, watermark, OCR)
  - Audio Studio (multi-track DAW with real-time waveform recording)
  - YouTube Downloader (via cobalt.tools, Y2Mate, SaveFrom)
  - Chemistry tools (equation balancer, pH calculator, molecular visualizer)
  - Research paper finder, unit converter, periodic table
- **E-Reader** — Document viewer with markdown and math rendering
- **3D Molecule Viewer** — Interactive molecular visualization
- **Responsive Design** — Works on desktop, tablet, and mobile

## 🛠 Tech Stack

- **Static Hosting:** GitHub Pages
- **Build Tool:** Vite 6
- **Frontend:** HTML, CSS, JavaScript (vanilla + ES modules)
- **3D Graphics:** Three.js, 3Dmol.js
- **Diagramming:** Mermaid.js
- **Audio:** Web Audio API, MediaRecorder API
- **PDF:** pdf-lib, pdf.js, tesseract.js
- **Icons:** Lucide
- **CI/CD:** GitHub Actions (auto-deploy on push)
